Crunch

Charactar set file /usr/share/crunch/crunch.lst

Syntax: crunch <min-len> <max-len> [<charset string>] [options]

Ex: 

  • crunch 3 4 -o outputfile.txt
  • crunch 3 4 abcdE -o outputfile.txt
  • crunch 3 4 -f /usr/share/crunch/charset.lst hex-lower -o outputfile.txt
  • crunch 5 5 -t @@@Ab -o outputfile.txt
    • @ will insert lower case characters
    • , will insert upper case characters
    • % will insert numbers
    • ^ will insert symbols
  • crunch 5 5 -t @@@Ab -s bbbAb -o outputfile.txt   //start from
  • crunch 8 8 -t ,@%admin -s cC5admin -o outputfile.txt
