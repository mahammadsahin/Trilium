File Permission

File Permissions

  • Read - r
    • Open and read
  • Write - w
    • Modify the contents
    • Add, remove, & remane
  • Execute - x
    • run
  • First - is used for file - -rwxrw-r--
  • Absolute (Numeric) Mode
    • 0 for no permission
    • 1 for execute
    • 2 for write
    • 4 for read

 

  • Symbolic Mode (for all 3 user types)
    • Relative 
      • + for adding permission
      • - for removing permission
      • = sets & overrides permission

 

  • U - User/owner
  • G - group
  • O - other
  • A - all

 

[*] File permission
