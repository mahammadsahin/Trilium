Transport Layer Protocol: Transport Layer in computer networks is an end to end layer which is used to deliver data to a host. Data communicate between Application layer to Network Layer.

The two most commonly used transport layer protocols are the connection oriented TCP & connection less UDP

 

Transport layer provides transparent transfer of data between end users, providing reliable data transfer services to the upper layer. the transport layer controls the reliability of a given link through flow control, segmentation & desegmentation and error control. Responsibilities of Transport layer: Process to process delivery, End-to-end connection between hosts, multiplexing & demultiplexing, congestion control, data integrity & error correction, flow control

TCP: TCP stands for Transmission control protocol is a connection-oriented transport layer protocol
