Linux User Administration

  3. Other

[*] when create a user that was multiple id - (i) parmanent id (uid) (ii) parmanent group id (gid) (iii) groups

 

User Types

  1. Administrator - Root (User Id=0)
  2. General User - Made with useradd commands (User Id=500-60000)
  3. System User - Made Internally (User Id=1-1499 & 60001-65535)

[*] Any user who will be given User id as 0 will become root or administrator and get full access or rwx

 

Permissions of Files

  • Read - r
    • Open and read
  • Write - w
    • Modify the contents
    • Add, remove, & remane
  • Execute - x
    • run
  • First - is used for file - -rwxrw-r--
  • Absolute (Numeric) Mode
    • 0 for no permission
    • 1 for execute
    • 2 for write
    • 4 for read

 

  • Symbolic Mode (for all 3 user types)
    • Relative 
      • + for adding permission
      • - for removing permission
      • = sets & overrides permission

 

  • U - User/owner
  • G - group
  • O - other
  • A - all
